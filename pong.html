Ping Pong Game

This was my first project created at Hackbright Academy in 2016.

For this project I wanted to recreate a ping pong tutorial. My goal was for it to be interactive and via a mobile device such as an Android.

Kivy allowed me to create such a project and also had a tutorial that enable me to launch it via the Kivy Launcher. I enhanced the tutorial further by adding sound and background music as well as color, asn transition from start to end.

Tutorial Website: https://kivy.org/docs/tutorials/pong.html
